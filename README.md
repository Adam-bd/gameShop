# Game Store Web Application

A web application built with Java and Spring Boot that simulates an online game store. The system provides role-based access control with separate functionalities for users and administrators.

## Features

### User

* Registration and authentication
* Browsing available games
* Adding games to a shopping cart
* Placing orders
* Viewing order history and order statuses

### Administrator

* Adding, editing, and removing games from the catalog
* Managing customer orders
* Updating order statuses (e.g. Order Placed → Processing → Shipped → Delivered)
* Managing store inventory

## Technologies

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* Hibernate
* MySQL/PostgreSQL
* Thymeleaf (or React/Angular if applicable)
* Maven

## Project Goal

The purpose of this project is to demonstrate the development of a full-stack web application using Spring Boot, including authentication, authorization, CRUD operations, shopping cart functionality, and order management.
